Multi-layer progression game: QuarkCore → Atomica → Bondia → Reaxia.
Current focus: **Bondia** (molecular construction).

**Why:** "Build. Discover. Understand. Repeat." — players reconstruct reality without feeling like they're learning.

**Core loop:** Collect/Observe → Combine/Build → Evaluate Outcome → Unlock → Apply Knowledge

**Molecule alignment system:**
- Beneficial: H₂O (healing), O₂ (life)
- Hazardous: CO (invisible, lethal), CH₄ (explosive)
- Situational: same molecule = save or destroy depending on context

**Key mechanics:**
- Energy: bonding consumes, completing molecules restores
- Stability: wrong structures decay/fail
- Hazard: toxic molecules harm player over time, chain reactions
- Environment reacts: oxygen-rich = stronger explosions, CO₂ buildup = suffocation

**Progression structure (Bondia):**
1. Simple two-atom bonds
2. Multi-element builds
3. Environmental awareness
4. Risk vs reward
5. Multi-step planning

**God Particles:** rare, multi-step, high-impact constructs that feel mythical.

**Learning philosophy:** teach through interaction/consequence, never explicit. Each molecule has a real-world fact embedded in its gameplay effect.

**Long-term:** full periodic table → advanced chemistry → physics → biological molecules → planetary simulation.

**How to apply:** Every feature added should have a purpose, a consequence, and a hidden lesson. No mechanic without meaning.
